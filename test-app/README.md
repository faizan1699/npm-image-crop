# Test App for React Image Cropper Pro

This is a test application to verify all features of the React Image Cropper Pro package.

## Quick Start

1. **Build the main package first:**
   ```bash
   cd ..
   npm run build
   cd test-app
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to the URL shown in the terminal (usually `http://localhost:5173`)

## Features to Test

- ✅ Image upload
- ✅ Drag to move crop area
- ✅ Resize handles (8 handles)
- ✅ Aspect ratio selection
- ✅ Crop functionality
- ✅ Download cropped image
- ✅ Reset functionality
- ✅ Auto-blob generation
- ✅ Console callbacks

## Troubleshooting

If you see import errors, make sure:
1. The main package is built (`npm run build` in parent directory)
2. Dependencies are installed (`npm install`)
3. The dev server is running (`npm run dev`)
